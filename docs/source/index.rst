Installation
============

The core of jatic-toolbox is lightweight: its only dependencies is ``typing-extensions``. 
To install it, run:

.. code:: console

   $ pip install jatic-toolbox

If instead you want to try out the features in the upcoming version, you can install 
the latest pre-release of jatic-toolbox with:

.. code:: console

   $ pip install --pre jatic-toolbox
