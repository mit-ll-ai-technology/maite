The Modular AI Trustworthy Engineering (MAITE) library provides common interfaces, utilities, and tooling to streamline the development of test and evaluation (T&E) capabilities, support core evaluation use cases, and allow composition of T&E tools into custom workflows.

Installation#

The core of MAITE is lightweight: its only dependencies are typing-extensions and numpy. To install it, run:

$ pip install maite

If instead you want to try out the features in the upcoming version, you can install the latest pre-release of MAITE with:

$ pip install --pre maite
